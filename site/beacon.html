Randomness Beacon

The randomness beacon provides a source of public randomness that is tamper-evident, ensured via hash-chain. Each minute a 512-bit (64-byte) pulse is published along with the previous pulse's hash, forming a tamper-evident chain. Find the latest pulse here, or search for a pulse by index or by (UTC) timestamp.
